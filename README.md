# Word Chain Animation

An interactive web application that displays a chain of Vietnamese words in an animated sequence with smooth transitions and visual effects.

## Description

This project creates a visually engaging animation that displays Vietnamese words one after another in a chain-like sequence. Each new word slides into view with a subtle animation effect while previously displayed words slide to the left.

Key features:
- Smooth animated transitions between words
- Audio feedback with a subtle "pop" sound when new words appear
- Interactive controls to start, pause, and reset the animation
- Side panel with a complete list of words that allows jumping to any point in the sequence

## Usage

1. Open the HTML file in any modern web browser
2. Use the controls at the bottom of the screen:
   - **Bắt đầu (Start)**: Begin or resume the animation
   - **Dừng (Pause)**: Pause the animation
   - **Chạy lại (Reset)**: Restart the animation from the beginning
3. Click any word in the side panel list to jump directly to that point in the animation

## Technical Features

- **Pure JavaScript**: No external libraries or frameworks required
- **CSS Transitions**: Smooth animations using CSS transitions and transforms
- **Dynamic Element Creation**: Word boxes are created and positioned dynamically
- **Web Audio API**: Used to generate the subtle pop sound effect
- **Responsive Design**: Adapts to different screen sizes
- **Vietnamese Language Support**: Fully supports Vietnamese text

## Customization

To customize the word list:
1. Open the HTML file in a code editor
2. Locate the `words` array in the JavaScript section
3. Replace or add entries to the array to create your own word chain

```javascript
// Vietnamese word chain
const words = ["xin chào", "chào bạn", "bạn bè"];
```
